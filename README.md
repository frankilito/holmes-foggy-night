# 福尔摩斯：雾都雨夜谜案 🔍🌧️

> 基于公共领域文学角色的独立再创作，与任何现代影视改编无隶属或背书关系。
> 角色、台词与案件仅取材于柯南·道尔原著及公共领域早期插画；模型、动画、音乐、音效全部原创。

1890 年代，暴雨笼罩的伦敦依然灯火通明。三起相隔数英里的爆炸案在同一分钟发生——迈克罗夫特判断有人正用巴贝奇密码筒协调全城犯罪。扮演夏洛克·福尔摩斯，自由攀爬屋顶、开启演绎视界、追查线索、击破莫里亚蒂的 55m 级机械扑翼机「莱辛巴赫乌鸦」，在午夜钟声前阻止全城爆破。

纯浏览器运行，无需安装。Three.js 旧版全局 API，零构建工具，零音频文件（WebAudio 全合成）。

## 👥 双人联机（福尔摩斯 × 华生）

1. 玩家 1 在标题页点 **创建双人案件**，得到 4 位案件码，分享给玩家 2。
2. 玩家 2 输入案件码点 **加入案件**——连上后玩家 2 扮演华生（单人模式中华生为常驻 NPC，联机时无缝接管）。
3. 主机权威模拟敌人/BOSS/案件关键证据；双方各自管理血量与背包。投弹、燃烧、关键证据、BOSS 弱点全部同步。
4. 网络建议：同 WiFi 或一方开热点体验最佳；已内置国内可用 STUN（小米/腾讯优先）+ TURN 兜底，跨网可穿。连不上时界面会给出分阶段提示。

## 🗺️ 系统清单

- **世界**：800m 解析地形（唯一碰撞源）、泰晤士河道、固定雨夜天气机（毛毛雨→暴雨→雷暴→雾歇）、体积雾分层、伦敦天际线远景（圣保罗穹顶/塔桥/议会大厦剪影）
- **城市**：480+ 连续街区立面、150+ 精确碰撞体、12+ 可进入门厅（含 221B 壁炉/化学桌/提琴/案件墙）、大本钟 96m 钟塔+脚手架登顶、考文特玻璃顶市场、歌剧院、苏格兰场、塔桥、车站雨棚；76 市民 + 16 马车 + 320 暖窗 + 煤气灯四层繁华
- **玩家**：第三人称、BOTW 式体力轮、攀爬/游泳/二段跳/丝翼滑翔/俯冲击/翻滚无敌帧、检查点、熔铁弹回、湿墙更滑
- **战斗**：4 武器槽耐久制（乌木手杖剑/破门锤/化学燃烧瓶/气压麻醉镖枪）、加固雨伞盾、演绎反击、克制系 VFX（月牙风压/地裂/蒸汽镖尾/积水冲击波）
- **演绎推理**：F 演绎视界（降饱和·细琥珀注释线·时间减缓）、12 线索点（4 关键证据→唯一推论→解锁 BOSS 三处机械弱点）、案件墙
- **敌人**：莫里亚蒂犯罪网络 5 种敌人 × 35 只（发条侦察鼠/炸药客/装甲猎犬/迷雾刺客/重装打手）+ 三阶段 BOSS「莱辛巴赫乌鸦」
- **NPC**：7 位原著角色（迈克罗夫特/华生/雷斯垂德/艾琳·艾德勒/哈德森太太/维金斯/猎犬托比），道具全部通过对话获得，哈德森太太金币商店
- **音频**：WebAudio 全合成 128 步侦探主题 BGM（贝克街/商业区/铸造厂/BOSS 四区域变体）+ 雨雷车马动态环境声 + 28 种合成音效
- **联机**：PeerJS P2P 主机权威（玩家 15Hz / 敌人 8Hz / BOSS 8Hz / 天气 4s + 事件即时可靠）

## 🔧 调试参数

`index.html?debug=1` 跳过标题开场、给全装备、暴露 `window.G/Player/World/Combat/Enemies/Story/Npc/Net`、显示 FPS。
可叠加：`&at=shrine|volcano|boss|tower` 传送、`&q=<quest>` 设任务、`&w=downpour` 天气、`&t=0.7` 时刻、`&nodraw=1` 只跑逻辑、`&mp=host|join&room=XXXX` 自动联机。
